## Known Issues / Decisions Needed

### Open Questions
1. **Duration Flexibility**: Is it strictly 2 hours, or can it extend to 4?
2. **Hands-on vs Demo**: What's the ratio of attendee exercises vs watching?
3. **Pre-requisites**: Should attendees have Cline installed beforehand?
4. **API Keys**: How to handle LLM API access for attendees during exercises?
5. **Venue/Format**: In-person, virtual, or hybrid?

### Potential Risks
- **Live Demo Failures**: API rate limits, network issues, unexpected AI behavior
- **Varying Skill Levels**: Some attendees more advanced than others
- **Time Management**: Demos can run long with Q&A
- **Environment Issues**: Attendees may have different VS Code/PowerShell versions

### Mitigation Strategies
- Pre-record backup demos
- Prepare multiple stopping points for different durations
- Create tiered exercises (basic → advanced)
- Provide VM/container option for consistent environment

# 2-Hour Version: "Core Concepts + Demo"

## Overview

| Attribute | Detail |
|-----------|--------|
| **Duration** | 120 minutes |
| **Format** | Half-day session, meetup, team introduction |
| **Breaks** | 1 x 10 min |
| **Live Demo** | 15-minute synthesis demo |
| **Hands-on** | None (audience watches demo) |

## Target Audience

Technical professionals who want to **understand and see agentic coding in action**. Covers all core concepts plus a meaningful live demonstration of the full agentic loop.
